# ⏱ Telegram-бот учёта рабочего времени склада

---

## 🚀 Пошаговая инструкция запуска (для начинающих)

---

### ШАГ 1: Создать Telegram-бота

1. Откройте Telegram
2. Найдите бота **@BotFather** (с синей галочкой)
3. Напишите ему `/newbot`
4. Введите **имя бота** (например: `Учёт времени склада`)
5. Введите **username** бота (например: `my_time_tracker_bot`) — должен заканчиваться на `_bot`
6. BotFather пришлёт вам **токен** — длинная строка вида:
   ```
   7123456789:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
   ```
7. **Скопируйте этот токен** — он понадобится на Шаге 4

---

### ШАГ 2: Настроить Google Sheets API

#### 2.1 — Создать проект в Google Cloud

1. Перейдите на [https://console.cloud.google.com/](https://console.cloud.google.com/)
2. Войдите в свой Google-аккаунт
3. Нажмите **«Select a project»** (вверху слева) → **«New Project»**
4. Назовите проект (например: `TimeTracker`) → нажмите **«Create»**
5. Убедитесь что проект выбран (его имя отображается вверху слева)

#### 2.2 — Включить API

1. В меню слева нажмите **«APIs & Services»** → **«Library»**
2. В поиске наберите **Google Sheets API** → нажмите на него → **«Enable»**
3. Вернитесь в Library
4. В поиске наберите **Google Drive API** → нажмите на него → **«Enable»**

#### 2.3 — Создать Service Account (сервисный аккаунт)

1. В меню слева: **«APIs & Services»** → **«Credentials»**
2. Нажмите **«+ CREATE CREDENTIALS»** → **«Service account»**
3. Введите имя (например: `timebot-service`) → нажмите **«Create and Continue»**
4. Роль можно пропустить → нажмите **«Continue»** → **«Done»**

#### 2.4 — Скачать JSON-ключ

1. На странице Credentials найдите свой Service Account в списке
2. Нажмите на его **email** (выглядит как `timebot-service@timetracker-123.iam.gserviceaccount.com`)
3. Перейдите на вкладку **«Keys»**
4. Нажмите **«Add Key»** → **«Create new key»**
5. Выберите **JSON** → **«Create»**
6. Файл скачается автоматически
7. **Переименуйте** скачанный файл в `credentials.json`
8. **Переместите** его в папку проекта `Time/` (рядом с `main.py`)

> ⚠️ **Важно:** Запомните email сервисного аккаунта (из пункта 2) — он понадобится в Шаге 3!

---

### ШАГ 3: Создать Google Sheets таблицу

1. Перейдите на [https://sheets.google.com/](https://sheets.google.com/)
2. Создайте **новую таблицу** (нажмите `+`)
3. Назовите её (например: `Учёт времени`)
4. **Скопируйте ID таблицы из URL:**
   ```
   https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBxjgmUUqptlbs74OgVE2upms/edit
                                          ^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
                                          ЭТО ваш GOOGLE_SHEETS_ID
   ```
5. **Откройте доступ сервисному аккаунту:**
   - Нажмите кнопку **«Share»** (зелёная, справа вверху)
   - В поле email вставьте **email сервисного аккаунта** (из Шага 2.4 п.2)
   - Выберите роль **«Editor»**
   - Нажмите **«Send»** (можно снять галочку «Notify people»)

> ✅ Лист `time_logs` создастся **автоматически** при первом запуске бота.

---

### ШАГ 4: Настроить файл `.env`

1. В папке проекта `Time/` найдите файл `.env.example`
2. **Скопируйте** его и назовите копию `.env`
3. Откройте `.env` в блокноте и заполните:

```env
BOT_TOKEN=7123456789:AAHdqTcvCH1vGWJxfSeofSAs0K5PALDsaw
GOOGLE_SHEETS_ID=1BxiMVs0XRA5nFMdKvBdBxjgmUUqptlbs74OgVE2upms
GOOGLE_CREDENTIALS_JSON_PATH=credentials.json
```

> Подставьте **свои** значения из Шагов 1 и 3!

---

### ШАГ 5: Настроить сотрудников и супервайзеров

1. Откройте файл `app/utils/roles.py`
2. Заполните **список сотрудников** — замените нулевые `telegram_id` на реальные:

```python
WORKERS = [
    {"id": 1, "telegram_id": 123456789, "name": "Иван", "role": "worker"},
    {"id": 2, "telegram_id": 987654321, "name": "Петр", "role": "worker"},
    {"id": 3, "telegram_id": 555555555, "name": "Сергей", "role": "worker"},
]
```

3. Добавьте **супервайзеров** — раскомментируйте и вставьте их Telegram ID:

```python
SUPERVISORS = {
    111222333: "supervisor",  # Ваш Telegram ID
}
```

> 💡 **Как узнать Telegram ID?** Напишите боту [@userinfobot](https://t.me/userinfobot) — он покажет ваш ID.

### ШАГ 6: Запуск бота

#### Вариант А — через Docker (рекомендуется)

```bash
# Откройте терминал в папке Time/
docker-compose up -d --build
```

Проверить что бот работает:
```bash
docker-compose logs -f
```

Остановить бота:
```bash
docker-compose down
```

#### Вариант Б — без Docker (локально)

```bash
# 1. Откройте терминал в папке Time/

# 2. Установите зависимости (один раз)
pip install -r requirements.txt

# 3. Запустите бота
python main.py
```

### ШАГ 7: Проверка

1. Откройте Telegram
2. Найдите вашего бота (по username из Шага 1)
3. Нажмите **«Start»** или напишите `/start`
4. Должно появиться меню с кнопками:
   - ▶️ Разгрузка
   - ▶️ Приемка
   - ▶️ Сортировка
   - ▶️ Размещение
   - ⏹ Завершить процесс
   - 📊 Мой статус
   - 📅 Мой день
5. Нажмите **«▶️ Разгрузка»** — бот ответит что процесс запущен
6. Откройте вашу Google Sheets таблицу — там появится запись!

## ❓ Частые проблемы

| Проблема | Решение |
|----------|---------|
| `Файл credentials не найден` | Проверьте что `credentials.json` лежит в папке `Time/` |
| `BOT_TOKEN не задан` | Проверьте файл `.env` — токен должен быть без кавычек |
| Бот не отвечает | Проверьте что вы правильно скопировали токен из BotFather |
| `Error 403` в Google Sheets | Убедитесь что вы дали сервисному аккаунту доступ к таблице (Шаг 3, п.5) |
| Групповые команды не работают | Ваш Telegram ID должен быть в `SUPERVISORS` в `roles.py` |
